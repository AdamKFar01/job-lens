# 🔍 JobLens

AI-powered job description analyser. Paste any job posting and get a structured
intelligence report: company research, role breakdown, red flags, fit assessment,
and tailored interview prep — all generated by a tool-calling agent backed by
Groq (LLaMA 3.3 70B).

## What it does

1. **Extracts** the company name and role from your pasted job description.
2. **Searches the web** for recent information about the company — culture signals,
   tech stack, Glassdoor sentiment, funding news.
3. **Analyses the JD** for required vs. nice-to-have skills, seniority level,
   and red flags.
4. **Produces a structured report** with six sections:

| Section | What you get |
|---|---|
| 🏢 Company Snapshot | What they do, size/stage, culture signals |
| 🎯 Role Breakdown | Must-haves, nice-to-haves, seniority level |
| 🚩 Red Flags | Honest assessment of warning signs |
| 🤝 Fit Assessment | Who would genuinely thrive here |
| 💬 Talking Points | Angles to emphasise in your interview |
| ❓ Questions to Ask | 3 tailored questions for the interviewer |

## Tech stack

- **Frontend:** [Gradio](https://gradio.app) (runs locally in your browser)
- **LLM:** [Groq API](https://groq.com) — `llama-3.3-70b-versatile` (free tier)
- **Web search:** [DuckDuckGo Search](https://pypi.org/project/duckduckgo-search/)
  with Wikipedia fallback
- **Architecture:** Tool-calling agent loop — the model decides when to search
  and when to write the final report

## Setup

**1. Clone the repo**

```bash
git clone https://github.com/your-username/job-lens
cd job-lens
```

**2. Install dependencies**

```bash
pip install -r requirements.txt
```

**3. Add your Groq API key**

Get a free key at [console.groq.com](https://console.groq.com), then create a
`.env` file in the project root:

```
GROQ_API_KEY=your_key_here
```

**4. Run the app**

```bash
python app.py
```

**5. Open your browser**

Navigate to `http://localhost:7860`.

## Notes

- Analysis takes **30–60 seconds** — the agent runs 2–3 web searches before writing
  the report.
- Groq's free tier has rate limits; the app handles 429s automatically with backoff.
- If DuckDuckGo search is unavailable, the agent falls back to Wikipedia for company
  research (less detailed but still functional).
